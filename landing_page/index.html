Key Features

GitHub Account Stats

Display basic information about the GitHub user, such as their username, avatar, follower count, and the number of repositories.

Repository Listing

Show a list of repositories owned by the user, along with relevant details like the number of stars, forks, and the primary language used.

Commit History

Display the recent commits made by the user, including commit messages, commit date, and linked repository.
