How to Use the Service

To access the services provided by this application, follow these steps:

  1. If you don't have an account, register here.
  2. If you already have an account, login here.
  3. After logging in, you can access your dashboard to view your GitHub statistics.
  4. You can also fetch GitHub stats for a specific username by visiting the GitHub Stats page.
  5. To log out, simply click the "Logout" link in your dashboard.
